# NIST Alignment

## Current public demo

The current public demo is centered on **ML-DSA aligned with FIPS 204**.

This means the signer direction is intentionally anchored on the current NIST post-quantum signature standardization path for ML-DSA.

## Current visible focus

- **FIPS 204 / ML-DSA** for post-quantum signatures

## Why this matters

The public demo is not just presenting generic “post-quantum” branding. It is intentionally built around the current NIST signature direction and makes that visible through:

- ML-DSA profile selection
- signer workflows
- key metadata
- key rotation and lifecycle handling

## Longer-term private direction

The broader private project may later extend this NIST-aligned direction with:

- **FIPS 205 / SLH-DSA** as an additional signature track
- **FIPS 203 / ML-KEM** for secure exchange / KEM-oriented workflows

## Important clarification

This public repository is not a certification claim. It is a public demo aligned with the NIST direction currently chosen for the signer domain.
